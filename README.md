# HumanResource

A starter Python project running on the `HR` conda environment.

## Quick Start

1. Activate environment:
   ```powershell
   conda activate HR
   ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
3. Run app:
   ```powershell
   $env:PYTHONPATH="src"
   python test.py
   ```
4. Run tests:
   ```powershell
   $env:PYTHONPATH="src"
   pytest -q
   ```
